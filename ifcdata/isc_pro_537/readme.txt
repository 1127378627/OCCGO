              画像サイズ一括変換 Pro シェアウェア版　Ver.5.3.7
--------------------------------------------------------------------------------
■概要

  画像のサイズを一括で変更するソフトです。
  BMP、JPEG、GIF、PMG、TIFF に対応しており、リサイズだけでなく画像フォーマットの
  変換やファイル名のリネイムもできます。
  また、イメージ一覧のホームページを自動作成する機能もあります。
  
  Ver.5より画像の余白を削除する機能が追加されました。
  メーカー、あるいは仕入先から提供される商品画像に余白がある場合
  その余白を取って、サイズ変更することが出来ます。

■特徴

  ・別のフォルダにある画像ファイルでも複数指定できます。
    読み込める画像は、BMP、JPEG、GIF、PNG、TIFF

  ・画像の余白を削除することができます。

  ・画像に文字や画像を挿入することができます。

  ・ファイル情報一覧とサムネイル一覧を切り替える事ができますので、内容の確認が
    容易にできます。
    またファイル情報一覧で列クリックによる並び替えや、サムネイル一覧でドラックによる
    順序変更ができます。
    サムネイルを作成して表示しますので、元ファイルはすぐに解放します。
    またデジカメで撮影した画像等サイズの大きいファイルでもあまりメモリーを消費し
    ません。

  ・同じ画像を左右に並べ、それぞれ変換方法を変えて比較し、最適な変換方法を決定す
    ることができます。

  ・画像を回転することができますので、デジカメ等で縦サイズに撮影した画像を90度回
    転させたり、上下が逆になった画像を180度回転させて正しい向きで保存できます。

  ・リサイズされたファイルの名前を指定した規則に従って一括で変更できます。
    個別に特定のファイルのみ変更することもできます。（リネイム機能）
    ファイルの順番を変更した上で、連番のファイル名にすることもできます。

  ・リサイズされたファイルの名前に文字を付加できます。（接頭語、接尾語）

  ・サイズは 1 ピクセル単位で指定できます。

  ・リサイズ時の補間方法を指定できます。
      既定                    Default              既定の補間モード
      低品質                  Low                  低品質補間
      高品質                  High                 高品質補間
      ニアレストネイバー      NearestNeighbor      最近傍補間
      バイリニア              Bilinear             双一次補間
      バイキュービック        Bicubic              双三次補間
      バイリニア(高)          HighQualityBilinear  高品質双一次補間
      バイキュービック(高)    HighQualityBicubic   高品質双三次補間
      バイキュービック(特殊)  HighQualityBicubic   高品質双三次補間

  ・リサイズ方法も 5 タイプから選べます。
      長辺を合わせる
        横縦いずれか長い方を指定サイズにし、短い方を同じ比率でリサイズ
      横幅を合わせる
        横幅を指定サイズにし、縦幅を同じ比率でリサイズ
      縦幅を合わせる
        縦幅を指定サイズにし、横幅を同じ比率でリサイズ
      正方形
        長辺を合わせてリサイズし、指定カラーの正方形の指定した位置に配置
      縦横サイズ指定
        長辺を合わせてリサイズし、指定カラーの指定したサイズの指定した位置に配置
        携帯やPHSの待ち受け画像等の作成時に有効です。

  ・保存形式は、BMP、JPEG（60 〜 99% で品質指定）、GIF、PNG、TIFF

  ・画像ギャラリーページを自動生成できます。
      サムネイルを一覧表示し、クリックで大きな画像をポップアップ表示するページです。
      ポップアップには Lightbox2 を使用します。


■起動

  ・EXEファイル、またはショートカットアイコンをダブルクリックして起動します。

  ・画像ファイルをEXEファイル、またはショートカットアイコンにドラックアンドドロッ
    プします。フォームが開いて一覧リストにドロップしたファイルがセットされます。
    複数の画像ファイルを一度に選択することができます。

  ・インストール時に「送る」にショートカットを作成した場合は、画像ファイルを右ク
    リックし、メニューの「送る」から「画像サイズ一括変換」を選択します。フォーム
    が開いて一覧リストにドロップしたファイルがセットされます。
    複数の画像ファイルを一度に選択することができます。

  ・コマンドラインで実行できます。
    指定したフォルダにある画像すべてを処理し、フォームを開かないで処理後終了しま
    す。
    バッチファイル作成画面で以下のコードを作成することが出来ます。

      ・コマンドライン用の文字列
      ・VBS ファイル
      ・VB、VBA モジュール
      ・VB.NET サンプルコード

■動作環境

  ・.NET Framework 4 がインストールされているPC
  ・Windows XP SP2 / Windows Vista / Windows 7（32bit版・64bit版） にて動作確認
    （64bit環境では、WOW64(32bitエミュレーター)上で作動します）

  このソフトは VB.NET で作成していますので、実行には.NET Framework 4 が必要です。
  インストールされていない場合は、マイクロソフト社のダウンロードセンターから
  ダウンロードしてインストールしてください。

  .NET Framework 4 ダウンロード
  http://msdn.microsoft.com/ja-jp/netframework/ff687189.aspx
